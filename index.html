Welcome to School Task Assignment System

This system allows school administrators to assign tasks to teachers and allows teachers to manage and complete their tasks easily.

Key Features

  • Admins can create and manage tasks.
  • Teachers can view and complete assigned tasks.
  • Teachers can search tasks by priority.
